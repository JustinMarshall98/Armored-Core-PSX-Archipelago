# Armored Core Location Groups

## Ravens
Includes the locations for defeating all 5 ravens in missions
- Valkyrie in Attack Urban Center
- Nine-Ball in Destroy Floating Mines
- Fefnir in AC Battle(1)
- Kamui Mk.17 in Destroy Plane Computer
- Sledgehammer in Mop Up Chrome Remnants(2)

## Mail
All locations for reading Mail in game.

## Missions
All locations for Missions in game.
